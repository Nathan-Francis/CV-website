Real-Time Strategy (RTS) game in Unity

Designed using the Unity engine, and written in C#. The assets are either freely downloaded or purchased and licenced through the Unity Asset Store.

The game encompasses a home scene with various features such as selecting levels, inventory system, quest page, shops etc.

The game-flow requires the player to collect resources and build an army while defending incoming hordes. There is an enemy-spawner and wave-machine that will generate enemies throughout the level. When no enemies remain, the level is won.

Goose Chase Game in Processing

A simplistic game designed in Processing language. The player controls a goose using WASD keys, and is required to escape the red dots chasing the goose. If the player gets too close to the centre box, a new dot will spawn. The game is lost when the goose is caught.

2D Platformer Game in p5.js

A 2D platform based game designed in JavaScript using the p5.js library.

The game theme is to move across the map and reach the flag pole. There are coins to collect, life counter and a win/lose screen.
